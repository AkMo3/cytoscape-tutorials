Functional Enrichment Analysis

This protocol is designed as a walk-through tour of popular functional enrichment analysis tools, as a replacement for a hands-on tutorial. This protocol describes the use of three functional enrichment tools:

  • Enrichr
  • WebGestalt
  • Interactive Enrichment Analysis

Example Data

This workflow uses a lung cancer example dataset from TCGA, which compares lung cancer biopsies with normal tissue. The data has been processed as follows: To produce the inputs necessary for two tools, the summarized data has been further processed to produce three separate input files:

  1. Calculate log2 fold change and adjusted pvalue for cancer vs normal per gene from group average expression values.
  2. Filter for significantly up-regulated genes: log2 fold change => 2 AND adjusted pvalue <= 0.05; list of 367 genes
  3. Filter for significantly down-regulated genes: log2 fold change <= -2 AND adjusted pvalue <= 0.05; list of 516 genes
  4. Gene ranking calculated for the full dataset using sign(log2 fold change) * -log10(pvalue).

Example Data

The data pre-processing results in three files:

  • The first two files are lists containing the gene symbols of the significantly up- and down-regulated genes, respectively. This input can be used for Enrichr and Interactive Enrichment Analysis.
  • The third file contains the full dataset, annotated with Ensembl IDs and with the gene rank as the only data column. This input can be used for WebGestalt and Interactive Enrichment Analysis.

Enrichr

The Enrichr tool offers an easy-to-use interface for basic over-representation analysis of gene sets.

Before starting analysis, you can browse the available gene sets under Libraries in the top menu.

Notes go here

Enrichr: Gene Set Libraries

Enrichr includes over 200 gene set libraries, including Gene Ontology terms, pathways, disease-associated sets, cell type markers etc.

You can also search for gene sets either by the term name under Term search or by gene under Gene search.

Notes go here

Enrichr: Start Analysis

Analysis is started by simply copying the list of genes from the text file, for example the list of significantly up-regulated genes, into the input box on the right of the Analyze tab and clicking Submit.

Note that this corresponds to the example data file containing up-regulated genes.

Enrichr: Table View

In addition to the default bar chart, results are also available as a table under Table. The table include the four enrichment scores calculated by Enrichr: P-value, Adjusted p-value, Odds Ratio and Combined score. More information about these scores can be found in the Enrichr documentation.

At the bottom of the table there is a link to Export entries to table which will download a .txt file with the full results table.

WebGestalt

WebGestalt offers more advanced analysis options for a smaller set of gene set libraries. Using WebGestalt, it is possible to run either Over-Representation Analysis(ORA), Gene Set Enrichment Analysis (GSEA) or Network Topology-based Analysis (TNA).

Notes go here

WebGestalt: Data source

WebGestalt data source includes Gene Ontology terms, pathways, networks, disease-associated sets, drug terms etc.

Notes go here

WebGestalt: Start Analysis

To start analysis, the Basic parameters section needs to be complete in terms of Organism of interest, Method of Interest (ORA, GSEA or NTA) and Functional Database. The functional database chosen can be further specified, for example choosing pathway -> Wikipathway .

Notes go here

WebGestalt: Start Analysis

The input data is defined in the Gene List section, and the dataset or list of genes can either be pasted into the input field or uploaded as a text file. Select Gene ID Type should match the identifier type in the input data. The list used here corresponds to the example input file containing the full dataset with associated gene rank.

Clicking Submit starts the analysis.

Notes go here

WebGestalt: Results Summary

The top of the results page includes a job summary and a link to download the full results.

WebGestalt: Enrichment Results

Results are displayed as a bar chart by default. Right-clicking on the bar chart lets you download in either PNG or SVG format.

Clicking on the bars updates the pathway-specific display at the bottom of the page.

WebGestalt: Enrichment Results

The gene set (pathway in this case) specific results include the scoring statistics calculated for the enrichment, including the enrichment score, a sortable table and the enrichment plot.

The enrichment plot is described in detail here. Briefly, the plot in the upper half represents the running enrichment score (ES), as the analysis walks down the ranked list of genes. The middle section shows where the members of the gene set appear in the ranked list of genes. The top score in the enrichment map plot is the score reported for the particular gene set.

WebGestalt: Enrichment Results

For WikiPathways results, the link in the upper left is clickable and will open a pathway view with the overlapping/leading edge genes highlighted.

WebGestalt: Enrichment Results

In addition to the bar chart, the main results overview can also be visualized as a table or volcano plot by clicking the Table and Volcano buttons at the top left.

The volcano plot has options for customized downloads available, as well as pan/zoom controls.

Interactive Enrichment Analysis

Interactive Enrichment Analysis is a user-friendly interactive tools to perform enrichment analysis for multiple datasets across multiple public databases. This tool can run both Over-Representation Analysis(ORA) and Gene Set Enrichment Analysis (GSEA).

Notes go here

Interactive Enrichment Analysis: Database

A set of public database collections is provided in the drop-down in the Database section. These are updated monthly and datestamped. It is also possible to construct your own database collection under BUILD NEW DATABASE, and providing one or more GMT files.

Once a database collection is selected, the specific databases contained in the collection will be displayed.

Notes go here

Interactive Enrichment Analysis: Datasets

Datasets are selected in the Datasets section; one or more datasets can be selected if they are all the same format. A column named gene is required, and additional columns named rank, p.value, and fold.change are allowed and necessary for GSEA.

Notes go here

Interactive Enrichment Analysis: Dataset Options

The tool will display a preview of the first few rows of the first chosen dataset along with the required and optional columns that were detected.

Identifiers and organism should match the data. The analysis parameters can also be adjusted here.

Notes go here

Interactive Enrichment Analysis: Start Analysis

When Databases and Datasets are loaded and verified, the side panel will be updated with a Run Analyses option.

The initial set of panels will collapse and a Progress panel will open along with a progress bar in the lower-right corner. The Progress panel will report the individual analysis steps, any errors, as well as start and finish times for the run, and the location of the output folder.

Notes go here

Interactive Enrichment Analysis: View Results

Analysis results can be viewed by clicking View Results. Results are viewed in another app that allows you to interactively explore results and download individual files.

The Data tab displays the processed input data as a table and volcano plot. The table is paginated and searchable. A checkbox at the top allows you to also view the genes that were excluded from the analysis, e.g., due to failed identifier mapping. Both the table and plot are downloadedable.

Notes go here

Interactive Enrichment Analysis: Data plots

A volcano plot of your input data highlights subsets of genes based on the p.value and fold-change cutoffs indicated during the run. By default, the volcano plot uses the user-specified fold.change and p.value cutoffs (for ORA)

A bar plot of your input data highlights positive and negative fold-change values for a subset of genes.

Notes go here

Interactive Enrichment Analysis: Analysis Results

The Results tab displays a table of enriched terms/pathways, a plot of top results, and a plot of a selected result from the table. The table is paginated and supports selection, search and filtering. The plots for top results include dot, emap, concept networks, etc.

The plots for individual, selected results include GSEA score, database-specific views and linkouts, etc, and the results table can all be downloaded.

Notes go here

Interactive Enrichment Analysis: Analysis Results

Using the side panel, one can navigate between GSEA and ORA methods while viewing the results for a particular database to explore hits in common (and unique) to the different methods.

Notes go here

Interactive Enrichment Analysis: Dot plot

A dot plot of results highlight multiple statistics for the most enriched results. The plot has many options, including how many results to include, x-axis parameter, dot sizing parameter, and label and color options.
