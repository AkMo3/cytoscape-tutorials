Advanced Visualization: Tuning Layouts

This short protocol reviews how to tune layout algorithms in Cytoscape.

Tuning Layouts

In general, most layouts can be tuned to improve the aesthetics. My approach:

  • Go to Layout→Settings. Select the layout of choice, for example Prefuse Force Directed Layout.
  • Change a parameter, for example Default Spring Coefficient or Default Spring Length.
  • Click Apply Layout.
  • Observe the results. If you get the result you want, click Done, otherwise click Edit→Undo and apply the layout again.
