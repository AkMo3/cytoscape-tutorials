Enrichment Analysis and Cytoscape

  • Enrichment based on nodes in a network
  • All current enrichment apps do ORA

Apps for enrichment analysis

EnrichmentTable
Now a core app. Provides basic enrichment functionality.
stringApp
Enrichment data downloaded with STRING networks.
BinGO
Original enrichment app for Cytoscape. Creates a term network.
ClueGO
Most popular Cytoscape app. Will also do comparative enrichment. Creates a term network.
EnrichmentMap
Visualizes enrichments of terms as an enrichment map, a network representing overlaps among enriched terms.

Apps for enrichment analysis

EnrichmentTable

  • Core app (included in the release) in Cytoscape 3.10.
  • Uses g:Profiler to calculate the enrichment
  • Includes chart visualizations
  • Can have whole-genome background or current network background
  • Selection in Enrichment Table will select nodes in network

Apps for enrichment analysis

stringApp

  • Second most downloaded Cytoscape app
  • Enrichment bundled with stringApp
  • Includes chart visualizations
  • Can have whole-genome background or select any network as a background
  • Can do enrichment of clusters in a network
  • Selection in Enrichment Table will select nodes in network

Apps for enrichment analysis

BinGO

  • Original enrichment app/plugin for Cytoscape
  • Provides the ability to import alternative ontologies and annotations
  • Provides a nice network-display showing the terms, enrichment, and connectivity

Apps for enrichment analysis

ClueGO

  • Most downloaded Cytoscape app
  • Can import alternative ontologies and annotations
  • Produces nice network of terms, including visualization of multiple clusters
  • Can change background, but must provide separate reference set to do so

Apps for enrichment analysis

EnrichmentMap

  • Accepts enrichment data from other apps (stringApp and EnrichmentTable) and pipelines to produce a "map" of terms and term relationships
  • Can also take data from GSEA for ranked gene analysis
