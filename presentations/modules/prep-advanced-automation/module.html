STEP 2: Install the basics

    R users:

  1. Install R (update if over a year old)
  2. Install RStudio
  3. Install Bioconductor's manager: 
    
        install.packages("BiocManager")
        BiocManager::install()

    Python users:

  1. Install Python 3
  2. Install pip (if not included)
